# Chart-Horizon.com

Static landing page for ChartHorizon, served at **chart-horizon.com** via GitHub Pages.

- `index.html`, `impressum.html`, `privacy.html` — the pages
- `assets/` — images (dashboard screenshot, favicon)
- `CNAME` — pins the custom domain (do not delete)
- `.github/workflows/deploy-pages.yml` — deploys the repo root to GitHub Pages on push to `main`

Local preview: `python3 -m http.server 8080` then open <http://localhost:8080>.
